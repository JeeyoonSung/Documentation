# Coding Convention

TWTL Doc 003

TWTL 프로젝트에서 작성하는 프로그램 소스 코드에 대한 규약을 기술합니다.

## Overview

* Engine

 기본적으로 윈도우 API 규약을 따른다.
 
* GUI

 기본적으로 C# Standard 규약을 따른다.


### Coding convention elements

1. 들여쓰기

 절대 space와 tab을 섞어쓰지 않는다. 들여쓰기는 tab으로 통일한다.
 
1. 문장의 종료

 문장의 종료는 세미콜론(;)을 사용한다.
 
1. 명명규칙

 * 공백을 허용하지 않는다.
 
 * 상수는 대문자와 언더스코어
